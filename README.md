E-Commerce Web Application

Full-stack e-commerce website built with PHP, MySQL, JavaScript, HTML, and CSS. This project demonstrates essential web development concepts including database management, session handling with cookies, and dynamic content rendering.

- Core Functionality -

Product Catalog - Browse products organized by categories
Category Navigation - Easy navigation through 10 product categories
Shopping Cart - Add products to cart with cookie-based storage
Product Search - View products filtered by category
Responsive Design - Modern glassmorphism UI with gradient backgrounds
Dynamic Content - Database-driven product and category display

- Technical Features -

PHP backend with MySQL database
Cookie-based cart management (no user accounts required)
Prepared statements for SQL injection prevention
Responsive grid layouts

- Usage -
Browsing Products

Homepage - View all product categories
Click a category - See all products in that category
Add to cart - Click "Add to Cart" button on any product
View cart - Click cart icon in header
