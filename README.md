 - E-Commerce Web Application

Full-stack e-commerce website built with PHP, MySQL, JavaScript, HTML, and CSS. This project demonstrates essential web development concepts including database management, session handling with cookies, and dynamic content rendering. <br>

- Core Functionality <br>

Product Catalog - Browse products organized by categories <br>
Category Navigation - Easy navigation through 10 product categories <br>
Shopping Cart - Add products to cart with cookie-based storage <br>
Product Search - View products filtered by category <br> 
Responsive Design - Modern glassmorphism UI with gradient backgrounds <br>
Dynamic Content - Database-driven product and category display <br>

- Technical Features <br>

PHP backend with MySQL database <br>
Cookie-based cart management (no user accounts required) <br>
Prepared statements for SQL injection prevention <br> 
Responsive grid layouts <br>

- Usage <br>
  
Browsing Products <br> 

Homepage - View all product categories <br>
Click a category - See all products in that category <br>
Add to cart - Click "Add to Cart" button on any product <br>
View cart - Click cart icon in header <br>
